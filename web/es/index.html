Introducción

Escuela de Scala comenzó como una serie de presentaciones en Twitter para preparar a ingenieros expertos en programadores de Scala productivos. Scala es relativamente un lenguaje nuevo, que hace uso de conceptos familiares. Así, estas presentaciones asumieron que la audiencia estaba familiarizada con esos conceptos y mostramos cómo usarlos en Scala. Encontramos que es una forma muy efectiva para que los nuevos ingenieros cojan velocidad pronto. Este es el material por escrito que acompañó dichas presentaciones. Creemos que es útil en si mismo.

Enfoque

Pesamos que la mejor forma de enseñar Scala no es como una versión mejorada de Java sino como un nuevo lenguaje. No se asume experiencia previa en Java. Nos centraremos en el intérprete y el estilo objeto-funcional así como el estilo de programación que hacemos aquí. Pondremos especial interés en mantenibilidad, claridad en la expresión y el sistema tipado.

Muchas de las lecciones no requieren más software que el intérprete de Scala, REPL. Animamos al lector a seguir, pero también a ir más allá! Use las lecciones como el punto de partida para explorar el lenguaje.
